进程

1.查看进程

ps:执行ps命令的那个时刻的那些进程

参数：

-e 显示所有进程。

-f 全格式。

显示所有进程信息，连同命令行:ps -ef

结果的字段有：

  • 1.UID       用户ID
  • 2.PID        进程ID
  • 3.PPID      父进程ID
  • 4.C           CPU占用率
  • 5.STIME     开始时间
  • 6.TTY         开始此进程的TTY—-终端设备
  • 7.TIME       此进程运行的总时间
  • 8.CMD       命令名

通过进程名查看进程: ps -ef | grep 进程名

htop：动态的查看进程

安装:apt-get install htop，查看进程：htop

2.查看网络相关信息

netstat:一般用于检验本机各端口的网络连接情况。

参数：

-a : 显示所有的连接 -n : 以ip格式显示 -p : 显示连接对应的进程

字段部分含义：参考资料

通过进程号查看端口等信息 netstat -nap | grep 进程pid

alias name='command line'
