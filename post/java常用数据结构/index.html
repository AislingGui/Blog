Set

Set是java中的集合类型,三种常见的Set,HashSet,TreeSet,LinkedHashSet。 如果你需要一个访问快速的Set，你应该使用HashSet；当你需要一个排序的Set，你应该使用TreeSet；当你需要记录下插入时的顺序时，你应该使用LinkedHashSet。

1
2
3
4

 //定义String类型的Hashset,用父类Set声明，子类HashSet实例化
 Set<String> hashset = new HashSet<String>();
 //也可写成如下形式
 HashSet<String> hashset = new HashSet<String>();

另外两种集合定义：

1
2

 Set<Integer> treeSet = new TreeSet<Integer>(); //Integer类型的TreeSet
 Set<Integer> linkHashSet = new LinkedHashSet<Integer>();//Integer类型的LinkedHashSet

Map

Map是一种依照键（key）存储元素的容器。Map中不能有重复的键（Key），每个键（key）都有一个对应的值（value）。

定义HashMap

1

 HashMap<Integer, Integer> map = new HashMap<Integer, Integer>();

Stack

定义：

1

Stack<Integer> = new Stack<Integer> date;

常用方法：

  • push(E item):把元素压入栈顶
  • pop()移除堆顶元素，并作为此函数的返回值。
  • empty():判断栈是否为空
  • peek()：查看堆顶元素，但不移除。
  • search(Object o),返回对象在堆栈中的位置，以1为基数
